# Programadores para la Paz – Semana 8

## Actividad

Esta semana se trabaja en los temas de seguridad y en la revisión del sistema presentado en los archivos de la carpeta `semana8`.

## Objetivo

Revisar y documentar aspectos de seguridad de plataformas, describir el sistema y luego contestar las preguntas de la semana.

## Archivos de la semana 8

- `preguntas-semana8.txt`
- `reflexion-semana8.txt`
- `seguridad-plataformas.txt`
- `sistema.txt`
- `seguridad/` (carpeta con buenas prácticas de seguridad)

## Recomendaciones

1. Lee los archivos de contenido.
2. Completa las preguntas en `preguntas-semana8.txt`.
3. Redacta tu reflexión en `reflexion-semana8.txt`.
4. Revisa las buenas prácticas en `seguridad/`.

## Evidencia

La entrega se realiza con el commit y push de los cambios en tu rama personal del repositorio.

## Flujo de trabajo

1. Asegúrate de trabajar en tu propia rama.
2. Edita los archivos dentro de `semana8`.
3. Revisa los cambios con `git status`.
4. Haz `git add .` y `git commit -m "Entrega semana 8"`.
5. Envía la rama a GitHub con `git push -u origin nombres_apellidos`.
