# Semana 6 - Guía de ejercicios y aplicación práctica

## Descripción

Esta actividad nos ayuda a poner en práctica los conceptos de Node.js · Express · JSON · req.body · Peticiones POST. Ayudándonos a cómo recibir y procesar datos enviados por el usuario.

## Objetivo

Comprender la importancia de la recolección organizada de datos y la automatización de procesos mediante herramientas de integración digital.

## Funcionamiento

1.Recepción de datos mediante la ruta /registro.
2.Procesamiento de información enviada en formato JSON.
3.Simulación del registro de incidencias comunitarias mediante la ruta /incidencia.
4.Respuesta automática del servidor con los datos recibidos.

## Tecnologías Utilizadas

*Node.js
*Express
*JSON
*Git y GitHub

## Consideraciones

Esta práctica permitió comprender cómo las aplicaciones pueden recibir información enviada por los usuarios, procesarla de forma organizada y responder utilizando datos estructurados. Además, se evidenció la importancia de JSON y req.body en el desarrollo de servicios web con Express.
