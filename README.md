# Programadores para la Paz

Proyecto de prácticas por semanas con ejercicios, servidores y actividades de aprendizaje.

## Estructura

- semana1/ … semana7/: carpetas de cada semana con ejercicios y recursos.

## Cómo probar

1. Instalar dependencias (si existe `package.json` en la semana correspondiente):

```bash
cd semanaX
npm install
```

2. Ejecutar servidor (si existe `server.js`):

```bash
node server.js
```
